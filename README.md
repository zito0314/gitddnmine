# gitddn

React + Vite + Ant Design 기반의 금융권 DevSecOps Governance Platform 데모입니다.

## Vercel 배포 설정

- Root Directory: `react-app`
- Build Command: `npm run build`
- Output Directory: `dist`

## Demo Accounts

- Admin: `admin@gitddn.local` / `admin`
- Internal User: `developer@gitddn.local` / `user`
- External Developer: `partner@gitddn.local` / `external`
